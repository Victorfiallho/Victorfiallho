# 👋 &nbsp;Hi there

I'm a software developer and AI engineer based in **Brazil**, first-semester student at the country's top AI program. I build automation systems that give humans their time back.

My focus right now: **AI-powered customer service for remote teams** — combining WhatsApp, LLMs, and workflow automation into products that actually ship.

&nbsp;

## 🎯 &nbsp;Current Project — AoTomate

I'm building **Hera Remoto** — a multi-tenant SaaS where an AI secretary handles WhatsApp conversations automatically, escalating to humans only when needed.

```
Customer texts → AI reads & responds → Human handles edge cases
```
